CSS - Cascading Style Sheet

selector { property: value; }

The priority of CSS in HTML is INLINE -> INTERNAL -> EXTERNAL


Inline CSS

Use "style" property inside a specific tag in the body of the HTML file

Internal CSS

Use "style" tag in head section of the HTML file

External CSS

Use "link" tag to connect external CSS file (include its location) to HTML file

Class vs Id Selector

- Class can be named more than once while Id is unique.
- Class is called by a dot '.', Id is called by a hashtag '#'.
- Any HTML elements (tags) can have more than one Class but only one Id.
- It is better to use Class as Selector than Id.
- Psuedo Classes is very useful.
